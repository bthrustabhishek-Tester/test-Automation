# Innoflex Test Automation\n\nThis repository contains UI test automation for the Innoflex login page using Playwright.\n\n## Installation\n\nInstall dependencies using npm:\n\n```bash\nnpm install\n```\n\n## Running Tests\n\nRun all tests:\n\n```bash\nnpm test\n```\n\nRun tests in headed mode:\n\n```bash\nnpm run test:headed\n```\n\nView test report:\n\n```bash\nnpm run report\n```\n
